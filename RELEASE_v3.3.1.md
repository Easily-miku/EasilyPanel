# EasilyPanel 3 v3.3.1

## 🚀 版本亮点

- **优化玩家检测机制**：不再频繁发送list命令，改为智能分析日志
- **减少控制台垃圾信息**：消除重复的"There are X/20 players online"输出
- **提高性能**：增加缓存机制，减少服务器负载
- **前端优化**：调整更新频率，提供更流畅的用户体验
